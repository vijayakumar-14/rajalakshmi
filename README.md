🎓 Rajalaskhmi Engineering College

This project is a modern, responsive website designed to showcase the authentic, everyday experiences of students at REC . Unlike traditional marketing sites, this platform focuses on natural photography and the true spirit of our campus.

📸 Key Features
Candid Gallery: High-quality, unposed photography of student life, lectures, and events.

Responsive Design: Fully optimized for mobile, tablet, and desktop viewing.

Interactive UI: Smooth transitions and a clean aesthetic to keep the focus on the visuals.

Fast Loading: Optimized image assets for a seamless browsing experience.

🛠️ Built With
Frontend: HTML5, CSS3, JavaScript 

Styling: Modern CSS layouts (Flexbox/Grid)

Deployment: [Vercel / GitHub Pages ]
